ralgos.generator
ralgos.generator.uuniform(p=104729, m=100, seed=None)
→ Generate a universal hash-based uniform random number generator.

Parameters:
    p: int
        A large prime modulus. Default 104729.
    m: int
        Output values lie in [0, m). Default 100.
    seed: int
        Seed for reproducibility.

Returns:
    Generator instance (uuniform)

Example:
    gen = uuniform(seed=42)
    gen.sample(3)  # [12, 45, 99]

ralgos.generator.kuniform(k, p=104729, m=100, seed=None)
→ Generate a k-wise independent uniform random number generator.

Parameters:
    k: int
        Independence degree.
    p: int
        A large prime. Default 104729.
    m: int
        Output range is [0, m). Default 100.
    seed: int
        Seed for RNG.

Returns:
    Generator instance (kuniform)

Example:
    gen = kuniform(3, seed=10)
    gen.sample(3)  # [14, 72, 33]
